1、了解 block 块
    盒子模型---DIV ，内容尺寸、边框、内部填充、盒子之间的间距
       可以控制盒子在界面上的视觉效果，位置。通过盒子控制 网页中某些内容的位置。
   
    block 块---普通的标签  <a>    <li>，让普通标签的 height  width 属性生效。
    

2、了解 CSS 有哪些高级技巧
   CSS2 版本是目前的通用版本，
        优点：
        1、所有浏览器都兼容
        2、针对特点浏览器做特定操作
        3、能够实现很多基本效果，
        缺点：
        1、动态交互性不强，鼠标移动、点击
        2、不能让浏览器进行硬件加速
        3、实现特效非常困难，圆角、阴影、变换
   CSS3
        优点：
        1、CSS 2 的缺点就是 CSS3 的优点
        缺点：
        1、不同浏览器兼容性不一样，针对不同浏览器，编写专门的样式
        2、比较耗费内存资源

3、CSS 简化
   相同的 页面效果，使用 更加简单的 html 和 css 代码来实现

   (1)布局，减少不必要的 DIV
   (2)简写CSS
   
   backgroud-rolor
   backgroud-image
   
   合在一起写
   
   background:


4、CSS DIV 和 SPAN
   DIV 盒子，本身在 HTML 代码中没有意义，但是在 CSS 中非常有用。
   SPAN 属性，专门用在文字中，强调某个文字比较特殊。本身也没有任何意义。
