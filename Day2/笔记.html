1、了解网页相关的技术

        网页技术：是一种通用的技术，能够实现在浏览器中展示文字、图片、

声音、视频、游戏、pdf文档、word文档、、、、、、、、


       生活中网页无处不在。网页技术历史：

      1990年开始，美国国内互联网的硬件已经建设完毕、W3C组织已经把

互联网上相关协议、技术已经制定完毕。全世界很多技术专家，开始研究

如何通过互联网传递更多的数据内容。传递数据的格式要求能够跨平台。

可扩展"标记"语言----XML。

      标记(标签)语言特点：

      (1)使用 <>

      (2)由 <开始标签>标签内容</结束标签> 三部分组成

      (3)可以在开始标签中，添加属性，增强标签的性能

                    <man  age="32"  height="170" >刘立立</man>

           一看到标签，能够让人知道里面的内容是神马意思。可以通过

属性进行扩展。

            属性：attributeName="value"

      (4)在上述基础条件下，可以对 标签进行变化。

             <man name="刘立立" age="32" height="170"  ></man>

             <man  name="刘立立"  age="32"  height="170"  />

     (5) 标签是可以嵌套的

           <men>

                 <man  age="32"  height="170" >刘立立</man>

                 <man  age="22"  height="160" >张敏</man>

                 <man  age="40"  height="180" >王刚</man>

          </men>


      编写标签语言的技巧：

       (1)在写开始标签时，同时写上结束标签；

       (2)要进行格式缩进； 


     

     HTML 是基于 XML 改版而成，强制规定了一套标签和属性，按照 HTML

规定编写代码，能够在所有的浏览器中运行。

     HTML(超文本标记语言)，说明 HTML 除了传递文字内容外，还可以

传递更多类型的内容，传输的内容类型由 MIME 协议决定。

    常见的MIME类型(通用型)：

    超文本标记语言文本 .html          text/html

    xml文档                       .xml           text/xml

    CSS 样式表                 .css            text/css

    javaScript脚本          .js                text/js



      

小结：

    html 90年代初诞生，解决在互联网上不同操作系统之间传递内容问题。

以 XML 语言为范例创建的。最大特点：兼容性特别强


2、 掌握最基础的 HTML 语言作用

        在网页开发的技术中包含：HTML、CSS、JavaScript、

        HTML:  为内容提供结构

        CSS:      为内容提供样式

        JavaScript:为内容提供动态交互


        HTML 为网页中的内容，提供了很好的结构定义，标明每个内容意义(标题、段落、内容、加粗显示、并列关系的内容、图片、)

       CSS     为网页上基本的内容，添加颜色、字体、背景等样式修饰

        JavaScript 为网页上的内容提供与用户的交互。让网页中的内容动起来。


       如：找女朋友：

      按 HTML 标准找，找四肢健全，身体健康的；

      按 CSS    标准找，找 会穿衣、打扮的；

      按 JavaScript 标准找，找 能歌善舞、能都很好与人互动的；


      <p font-family="宋体" font-size="20px">的顶顶顶顶顶</p>     


3、 掌握 HTML 基本结构

<!--  文档内容类型，作为第一句话，告诉浏览器下面是神马内容 -->

<!DOCTYPE html>


<!-- 整个网页的主干  -->

 <html>

           <!-- 代码中，告诉浏览器的内容，用户看不到 -->

<head>演员的化妆间</head>

          <!-- 代码中，给用户看的内容  -->

<body>演员的舞台</body>

 </html>    


我们是观众，只能看到 body 里面的内容
